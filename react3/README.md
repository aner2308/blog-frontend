# Bloggprojekt – Anton Eriksson

En enkel blogg byggd med **React + TypeScript**.  
Användare kan logga in och skapa, redigera samt radera inlägg.

## Funktioner

- Visa alla inlägg
- Visa enskilt inlägg
- Skapa nytt inlägg (inloggad användare)
- Redigera inlägg
- Radera inlägg
- JWT-baserad autentisering

## Tekniker

- React
- TypeScript
- React Router
- Context API
- CSS

## Kör projektet lokalt

git clone https://github.com/aner2308/blog-frontend.git
npm install
npm run dev
